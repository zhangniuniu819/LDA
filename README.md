### 引言：

  近年来，女性主义话语在全球范围内经历了显著的 “媒介化转向”，不再局限于学术圈层的理论探讨，而是依托书籍、电影、脱口秀等大众文化载体，成为公共话语与消费市场的核心议题。这一趋势背后暗藏双重张力：一方面，书籍的深度阐释、电影的视听叙事、脱口秀的即时互动与冒犯性，使不同媒介在建构女性主义话语时呈现出鲜明差异；另一方面，资本迅速捕捉 “她经济” 商业潜力，大量作品主动贴上 “女性主义” 标签，进而引发 “粉色资本主义” 争议 —— 资本介入究竟是推动女性主义内容革新，还是仅将其沦为营销噱头？   为回应上述核心问题，本研究以书籍、电影、脱口秀三类媒介中的女性主义题材作品及受众评论为数据基础，综合运用主题建模、情感计算、统计检验等计算人文方法，聚焦话语建构差异与作品评分对比展开系统分析，并在控制主创咖位的前提下，实证检验 “女性主义” 标签对作品评分的实际影响。

### 研究材料数据：

来源于豆瓣网采用网络爬取+手工提取结合的方式获取 325 本女性主义书籍（书名、作者、出版年、评分等）及 6649 条对应书评 217 部女性主义电影（片名、导演、上映年、评分等）及 8961 条对应影评 40 篇女性主义脱口秀文稿及2600条分句
